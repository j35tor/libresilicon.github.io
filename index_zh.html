Libre Silicon Alliance

给众人的开源半导体

我们正开发一种自由使用及开源的半导体制作工艺标准, 目的是提供一种快捷, 容易使用, 及价格相宜的制程. 开始制定流程时, 无需遵守特殊的保密协议 NDA, 即使在个人设施条件下, 仍可按设计制作. 我们的目标是打断现时非开源厂商垄断!

背后理念 更多资料 起步点 工艺流程

把市场开放

现时并没有价格相宜及简易的方式 来让中小企业,初创公司以及个人开发 '特殊应用积体电路' (ASIC) 并推出公开市场. 差不多全部的制程都是由大型半导体厂商拥有,而开发产品原型晶片需用上两万美元以上,及多月的等候才可实际测试晶片. 另一方面, 所使用的发开EDA 软件亦需要额外支付一般初创企业负担不起的高昂费用, 我们希望带来改变.!

开源的工艺

为了使半导体市场更自由化及避免被大厂商垄断, 需要推行一种标准化公开的制作工艺, 来方便设计图则的交流及检查. 这是现时我们努力的方向.

不同地域皆可进行的可重覆性

我们在工艺上指定相应的设计参数, 亦同时支援数码及模拟电路. 所产生的效果: 即使在某地方设计的ASIC (同时包括CMOS及模拟电路), 亦可在地球另一端在GitHub下载有关的设计并进行生产.

可核查芯片, 解决对保安的担忧

由于现时不少人对个别厂家的中央处理器中管理引擎的担忧, 通过使用公开的制程及技术, 晶片的图则可以在 GitHub 上面公开. 而公开的中央处理器平台亦随之然发展, 其公开的程度可推至 晶体管 层面. 从此再不可以隐藏 非相关的功能.

一个协作的平台

透过使用区块链技术, 可不经手来自然追踪各自有贡献的使用者, 并确保每个人可获得其开发ASIC 以及IP Block 知识产产权区块所付出的资源. 产品买家亦可收到订单后在公开市场向卖家订货, 所产生的效果是供求相方透过市场公开配对.

Getting started

For the ones eager to get started with the whole ecosystem, below there can be found all the information required to do so.

The EDA: QtFlow

The GIT repository for the EDA can be found under the following URL:
https://github.com/libresilicon/qtflow

Prebuilt Debian packages for the EDA can be found under the following URL:
https://download.opensuse.org/repositories/home:/leviathanch:/qtflow-debian

The LibreSilicon technologies

The GIT repository for the standard cells is being maintained by Hagen under (CMOS only and only 1um yet!):
https://github.com/chipforge/StdCellLib

The LibreSilicon process

The GIT repository for the process spec is being maintained under (format LaTeX):
https://github.com/libresilicon/process

The LibreSilicon mailing list

https://list.o2s.ch/mailman/listinfo/libre-silicon-devel
